# reliber-lp

自身が所属・運営しているバレーボールチーム「Re:Liber」の
紹介・参加募集を目的としたLPサイトです。

## URL
https://reliber-lp.netlify.app/

## 制作背景・目的
- チームの雰囲気を分かりやすく伝える
- 初心者・未経験者が安心して参加できるようにする
- SNS（Instagram）への導線を作る

## 工夫した点
- 情報量を整理したシンプルなLP構成
- Flexbox / Grid を使ったレイアウト設計
- スマートフォン表示を意識したレスポンシブ対応

## 使用技術
- HTML
- CSS
- JavaScript
